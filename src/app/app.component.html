A guide to getting your development environment setup for Angular, Ionic 4, Capacitor, and Firebase.
Any issues please report to the repo.
 First install node. https://nodejs.org/en/download/ Then install the Angular CLI globally. https://angular.io/guide/quickstart
npm install -g @angular/cli
Then install the Ionic CLI globally. https://beta.ionicframework.com/docs/installation/cli/
npm install -g ionic
Then create an ionic app. Choose an app name, a style, and make sure to enter type = angular.
ionic start [appName] [tabs | sidemenu | blank] --type=angular
Enter the folder of the app.
cd [appName]
Build the project at least once.
npm run build
Install Capacitor to the project locally. https://capacitor.ionicframework.com/docs/getting-started/with-ionic/
npm install --save @capacitor/cli @capacitor/core
According to the capacitor docs, you need to uninstall Cordova Splash.
npm uninstall --save cordova-plugin-splashscreen
Initialize capacitor for your project. Add an appId, such as com.ilmstudios.rapidstart
npx cap init [appName] [appId]
Install the Firebase CLI globally. https://github.com/firebase/firebase-tools
npm install -g firebase-tools
Initialize Firebase for your app. Follow the commands to enable the features you want. If you want to enable firestore you need to first enable it from the Firebase console online. When prompted to select a folder for build output, enter 'www'. Setup for single page application. Select to not overwrite the index.html that is there currently.
firebase init
Deploy your built app to firebase.
firebase deploy
To run your app locally, run
ionic serve
You're done!
